Descrição

HTML Viewer Q is a lite HTML file management tool.
You can import files via ""iTune File sharing"" to it.
And the tool will help you to edit & view them easily.


******************************************************
Feature
******************************************************
◆ iTunes file sharing support.
◆ Local HTML files and bookmarks management.
◆ Source text search.
◆ Tabbed Browsing.
◆ iCloud Drive Access.
◆ Syntax highlighting.
